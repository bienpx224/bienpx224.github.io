Unity Game

Kinh nghiệm phát triển, cải tiến với hơn 10 dự án game phát triển bằng Framework Unity với những tựa game nổi tiếng như Hero rescue, Water Sort, Monster Run, Candy Fruit, How to loot... Tích hợp quảng cáo Admob, FAN, Unity và xuất bản nhiều ứng dụng lên CHPlay và Apple store, thiết lập tối ưu chạy quảng cáo Google Ads đa quốc gia, mang lại nguồn doanh thu thụ động.

Java Android

Ngoài việc xây dựng phát triển ứng dụng Android bằng Java, tôi có 2 năm kinh nghiệm trong việc lập trình với JNI, NDK, third-party library, xây dựng các thư viện SDK như Player Livestream Video, PJSIP Call, Login VietID, Player Exo, Log,.. Giúp cải thiện tốc độ, tối ưu hiệu quả dự án, dễ dàng tích hợp mở rộng và nâng cao bảo mật cho ứng dụng.

Web Developer

Triển khai dự án web từ phân tích tới sản phẩm bằng PHP và NODEJS. Có kinh nghiệm phát triển cả phía server (PHP, NODE) và client (BLADE, JS, REACT) với các Database như MySQL, MongoDB, Redis,.. tích hợp socketIO, firebase. Một vài dự án đã hoàn thành như Thi trắc nghiệm, Đặt hàng China, Tam Bảo Investment, Mạng xã hội thú cưng, Trang quản lý dự án Ticket...
